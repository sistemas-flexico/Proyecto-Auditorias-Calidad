# Sistema de Gestión y Monitoreo de Auditorías de Calidad
Sistema gráfico desarrollado en PHP para la gestión y 
monitoreo en tiempo real de auditorías digitales de 
calidad integrado al ERP corporativo conectado a base 
de datos en Progress para el departamento de Producto 
Terminado en FLEXICO S. de R.L. de C.V.

## Características principales
- Gestión de clientes, modelos y empaques
- Exportación a PDF
- Panel de administración (menú del sistema)
